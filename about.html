Sobre o Projeto

Um mapa interativo listando as lojas de instrumentos músicais em atividade na cidade de Campinas-SP

O site foi desenvolvido utilizando HTML e CSS e o framework Materialize

O mapa interativo foi gerado utilizando a ferramenta My Maps do Google

Projeto criado em 2018 para a disciplina de Desenvolvimento Web, ministrada pelo professor André Bordignon para o quinto semestre do curso de Análise e Desenvolvimento de sistemas do Instituto Federal de São Paulo, campus Campinas.

Projeto sob licença MIT, com código fonte disponível para consulta e modificação no GitHub
